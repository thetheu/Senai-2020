Apresentação sobre testes:

-O que é teste (definição)
  Todo mecanismo que busca verificar ou provar a verdade. Exame crítico ou prova das qualidades de uma 
pessoa ou coisa.

-Porque ele é importante
  O teste de API é importante como qualquer outro tipo de teste de aplicativo, pois ajuda a garantir o funcionamento, 
o desempenho e a confiabilidade de diferentes aplicativos e sistemas baseados em dados. Além disso,
 quando analisamos as últimas tendências de teste de software, eles são uns dos mais importantes.
os testes de API estão na lista de prioridades para verificação.

-No que ele agrega
  Quando queremos agregar qualidade a um software necessitamos realizar bons testes com o intuito de melhorar cada vez o mesmo
e com as APIs não é diferente. O teste de API é uma parte muito importante do ciclo de vida de desenvolvimento tanto da API 
quanto do sistema como um todo. Além disso, se uma API for interrompida porque defeitos foram escapados, pode ocorrer de não apenas quebrar
um único aplicativo, mas também toda uma cadeia de processos de negócios.
